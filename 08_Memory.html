A branch of the New York Public Library

We were broke.

Watching eccentrics at the NYPL cost nothing, though. I think we had a slice from the kosher pizza place that night, fittingly, before going to see what remains the only informative talk I've ever witnessed. For what felt like an implausibly long time, Some Guy spoke about every single pizza box design in America.

The crowd was quietly appreciative but cranky all the same. The first question of the Q&A came from a rasping-voiced old Brooklyn type.

”You know,“ Old Brooklyn said. ”You know I heard pizzas got a lot of salt.“
